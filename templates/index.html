Suggest Words from Rack

Format guide
Rack tiles
Enter up to 7 letters (A–Z). Use ? for a blank tile (acts as any letter, scores 0 points).
Example: RSTLNEI  |  with blank: RST?NEI
Starts / Ends / Contains
Letters already on the board — do not need to be in your rack. Multi-letter ok.
Use | to list alternatives (OR): Y|L = word contains Y or L.
e.g. rack STLNEI + Contains Y|L finds words that pass through a board Y or L.
Length
Exact number of letters (2–7).
Pattern mask
Mask applied to the word shape. Literal letters are treated as board tiles (free — not from rack):
_ = one letter from your rack  |  * = zero or more letters  |  A-Z = board tile at that position
Example: _A__R = A and R on board, you supply the other 3 letters
Example: ____S = 5-letter word ending in board S

{# ── PATTERN ─────────────────────────────────────────────────────────────── #} {% elif mode == 'pattern' %}

Pattern Search

Find all CSW21 words matching a pattern — no rack needed.

Pattern format guide
_ (underscore)
Exactly one unknown letter. Use one underscore per unknown position.
Example: _T_W_ = 5-letter word with T at position 2, W at position 4
* (asterisk)
Zero or more letters — variable length gap.
Example: *ING = any word ending in ING
Example: *QU* = any word containing QU
A–Z
Literal letter — must appear exactly at that position.
Example: S*S = any word starting and ending with S

More examples:  ___ (all 3-letter words)  |  *T_W* (T, one letter, then W, anywhere)  |  ____S (5-letter words ending S)  |  ^QU* — use literal Q or Z for high-value setups

Phrase Builder

Enter letters and find all combinations of words that use every letter exactly — e.g. YOUIEVOL → I + LOVE + YOU.

How it works
Letters
Up to 15 letters A–Z (no blanks). The finder returns every set of valid words that uses all your letters exactly.
Min word length
Shortest allowed word (default 1 — includes common English words A and I).
Max words
Maximum words per phrase (default 4). Higher values slow the search.
Results
Words are shown in sorted order — rearrange them into the phrase you want.

Examples:  YOUIEVOL → I + LOVE + YOU  |  LINSERTS → LINTERS + S?  |  longer inputs may take a moment
